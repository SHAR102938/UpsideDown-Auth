# 🧠 UpsideDown Auth

A Stranger Things–inspired password strength analyzer that helps users create secure passwords with real-time feedback and visual indicators.

---

## 🚀 Features

* 🔍 Real-time password strength analysis
* 🔢 Checks for:

  * Minimum length
  * Uppercase & lowercase letters
  * Numbers
  * Special characters
* ⚡ Instant feedback on strength (Low → Strong)
* 🧮 Entropy-based security estimation
* 🎨 Stranger Things inspired dark UI (Upside Down theme)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Design:** Custom CSS animations & dark theme UI
* **Logic:** Password validation + entropy calculation

---

## 🎯 How It Works

The analyzer evaluates passwords based on:

* Length (recommended ≥ 12 characters)
* Character variety
* Randomness (entropy)

Based on these factors, it calculates:

* Strength Score
* Complexity Level
* Estimated Crack Time

## 💡 Security Tips

* Use at least 12+ characters
* Mix uppercase, lowercase, numbers, and symbols
* Avoid common words or personal information
* Use unique passwords for every account
* Prefer passphrases over simple passwords
